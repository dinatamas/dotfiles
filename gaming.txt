Gaming on Arch Linux
====================

I use Lutris with Wine, Vulkan, Proton for gaming on NVIDIA proprietary drivers.
Steam and Battle.net are also available and usable.

```
sudo pacman -S --needed nvidia-dkms nvidia-utils lib32-nvidia-utils \
nvidia-settings vulkan-icd-loader lib32-vulkan-icd-loader

sudo pacman -S --needed wine-staging giflib lib32-giflib libpng lib32-libpng \
libldap lib32-libldap gnutls lib32-gnutls mpg123 lib32-mpg123 openal lib32-openal \
v4l-utils lib32-v4l-utils libpulse lib32-libpulse libgpg-error lib32-libgpg-error \
alsa-plugins lib32-alsa-plugins alsa-lib lib32-alsa-lib libjpeg-turbo lib32-libjpeg-turbo \
sqlite lib32-sqlite libxcomposite lib32-libxcomposite libxinerama lib32-libgcrypt libgcrypt \
lib32-libxinerama ncurses lib32-ncurses opencl-icd-loader lib32-opencl-icd-loader libxslt \
lib32-libxslt libva lib32-libva gtk3 lib32-gtk3 gst-plugins-base-libs \
lib32-gst-plugins-base-libs vulkan-icd-loader lib32-vulkan-icd-loader

sudo pacman -Sy vulkan-icd-loader lib32-vulkan-icd-loader vulkan-intel \
lib32-vulkan-intel nvidia-utils lib32-nvidia-utils

# For the presence of vulkaninfo.
sudo pacman -Sy vulkan-tools

sudo pacman -Sy lutris

prime-run lutris

sudo pacman -Sy steam
# Steam > Settings > Steam Play: for all titles!

prime-run steam
```

  - At first run Lutris may be updating runtimes.

  - For some reason nvidia-dkms was already installed for me, not nvidia.

How to detect which driver is installed?
e.g. lshw | less and then search for /NVIDIA
Detect if it supports Vulkan:
https://wiki.archlinux.org/title/Vulkan
prime-run vulkaninfo
